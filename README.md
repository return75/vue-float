# vue-draggable-component

## installation
```
 yarn install vue-float
```

### usage:
```
<vue-float >
  <h1>this is vue float component</h1>
</vue-float>
```
.vue file:
```
  import VueFloat from 'vue-float'
  import "vue-float/dist/vue-float.css"
  ...
  export default {
        components: {
            VueFloat,
        },
  ...
```
